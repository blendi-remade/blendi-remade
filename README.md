# 👋 Hey, I'm Blendi

working as an engineer at [fal](https://fal.ai/)

*some of my favorite work is:*

[Sprite Sheet Creator](https://github.com/blendi-remade/sprite-sheet-creator): Sprite sheet generator for 2D pixel art characters with AI-powered character generation from prompts or images, supporting side-scroller and isometric RPG-style games.

[Falcraft](https://github.com/blendi-remade/falcraft): Minecraft Fabric mod that uses fal.ai to generate AI-created 3D structures from natural language prompts, converted into voxelated block-based models.

[Isometric City](https://github.com/blendi-remade/isometric-city-fal): Browser-based isometric city builder with AI-powered custom building generation. Describe a building and it gets created and placed in your city.

[Earth Cinema](https://github.com/blendi-remade/earth-cinema): Chrome extension that captures Google Earth 3D views and transforms them into cinematic images and videos using fal.ai.

[fal 3D Anything](https://github.com/blendi-remade/fal-3d-anything): Chrome extension that converts any web image into an interactive 3D model via right-click, powered by Hunyuan 3D and fal.ai.

[fal 3D Unreal](https://github.com/blendi-remade/fal-3d-unreal): Unreal Engine 5 plugin that generates 3D characters from text or images, automatically rigs and animates them, and enables real-time player control.

[WorldPlay](https://github.com/blendi-remade/fal-worldplay): Immersive 3D world and character generator. Create interactive environments from text or images and explore them with AI-generated rigged characters in real time.

[Banana Peel](https://github.com/blendi-remade/banana-peel): Collaborative image evolution platform where users upload images and the community generates AI-powered variations through text prompts, creating branching chains of transformations.

[DESI Universe](https://github.com/blendi-remade/desi-universe): Interactive 3D browser visualization of 1.4 million real astronomical objects from the DESI Data Release 1, explorable by spectroscopic redshift.
